STEP 1 · 프롬프트 복사
이 키움증권 MY페이지 계좌별 자산 화면에서 각 계좌의 총자산을 추출해서 아래 JSON 형식으로만 응답해줘 (JSON 외 텍스트 없이): { "date": "YYYY-MM-DD", "accounts": [ {"key": "kiwoom-overseas", "name": "해외", "balance": 숫자}, {"key": "kiwoom-obil", "name": "오빌", "balance": 숫자}, {"key": "kiwoom-practice", "name": "연습", "balance": 숫자}, {"key": "kiwoom-jasaju", "name": "자사주","balance": 숫자}, {"key": "kiwoom-byuldong", "name": "별동대","balance": 숫자}, {"key": "kiwoom-chobil", "name": "초빌", "balance": 숫자} ] } 계좌번호→key: 5211-3751→kiwoom-overseas, 5128-4051→kiwoom-obil, 5806-1320→kiwoom-practice, 6139-5535→kiwoom-jasaju, 5845-3671→kiwoom-byuldong, 5185-9447→kiwoom-chobil 화면에 보이는 계좌만 포함. balance는 총자산 원 단위 정수. date는 오늘 날짜.
🤖 Claude.ai 열기 ↗
💡 사용 방법
  1. 키움 앱 → MY페이지 → MY키움자산 → 계좌별 자산 스크린샷 준비
  2. 위 프롬프트 복사 후 Claude.ai에서 스크린샷 + 프롬프트 붙여넣기
  3. Claude가 반환한 JSON을 아래에 붙여넣기
