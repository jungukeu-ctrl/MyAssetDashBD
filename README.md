# 📊 My Asset Dashboard

> 개인 자산을 한 눈에 — GitHub Pages 기반 단일 페이지 자산 관리 웹앱

**🔗 접속 URL:** [jungukeu-ctrl.github.io/MyAssetDashBD](https://jungukeu-ctrl.github.io/MyAssetDashBD/)

---

## 개요

별도 서버나 앱 설치 없이 브라우저에서 바로 사용하는 개인 자산 통합 관리 도구입니다.  
부동산 · 금융자산 · 연금/IRP를 하나의 화면에서 관리하며, Firebase를 통해 기기 간 데이터를 자동 동기화합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **총 자산 현황** | 부동산 · 금융자산 · 연금/IRP 합산, 도넛 차트 시각화 |
| **목표 자산 진행률** | 총 목표 / 금융 목표 두 가지 Progress Bar |
| **키움 포트폴리오** | 6개 계좌 평가금액·투자금·수익률 카드 및 차트 |
| **월별 차트** | 12M / 24M / 전체 구간 전환 가능한 평가금액·수익률 추이 |
| **스냅샷 입력** | 토스·키움·연금 JSON 붙여넣기로 잔고 즉시 반영 |
| **이체내역 소급** | 키움/연금(개인연금저축·IRP1) 월별 투자금을 과거 데이터 보호하며 자동 갱신 |
| **엑셀 내보내기** | 월별 평가금액·투자금·수익률·스냅샷 4개 시트 다운로드 |
| **Firebase 동기화** | 데이터 변경 시 2초 후 자동 저장, 수동 동기화 버튼 제공 |
| **할 일 메모** | 드래그앤드롭 정렬 지원 Todo 리스트 |

|----|------|
| `asset-dashboard-v3` | 수동 입력 자산, 스냅샷 최신값 |

## 데이터 입력 워크플로우

```
📱 앱 화면 캡처
      ↓
🤖 Gemini Gem 업로드 (gemini.google.com/gem/...)
      ↓
📋 JSON 자동 출력
      ↓
💻 대시보드 해당 버튼 → 붙여넣기 → 적용
```

### 입력 경로별 갱신 대상

| 버튼 | 갱신 대상 | 비고 |
|------|-----------|------|
| 토스 JSON 붙여넣기 | `state[toss-*]` | |
| 키움 스냅샷 | `state[kiwoom-*]` + `eval[]` | |
| 키움 이체내역 | `invest[]` | 원본 거래내역 JSON 또는 집계 JSON, 계좌 선택 UI 제공 |
| 연금 스냅샷 | `state[pension-*]` + `eval[3/7/8]` | |
| 연금 이체내역 | `invest[3]` 또는 `invest[7]` | 개인연금저축 / IRP1 라디오 선택 |

## 기술 스택

- **Frontend:** Vanilla HTML / CSS / JavaScript (프레임워크 없음)
- **차트:** [Chart.js 4.4.1](https://www.chartjs.org/)
- **엑셀:** [SheetJS (xlsx) 0.18.5](https://sheetjs.com/)
- **데이터베이스:** [Firebase Realtime Database](https://firebase.google.com/)
- **호스팅:** GitHub Pages
- **이미지 추출:** Gemini Gem (커스텀)

---

## Firebase 설정

`js/config.js` 상단의 `FIREBASE_URL`을 본인의 Firebase Realtime Database URL로 교체하세요.

```js
const FIREBASE_URL = 'https://<your-project>-default-rtdb.<region>.firebasedatabase.app';
```

Firebase 콘솔에서 데이터베이스 규칙(Rules)을 읽기/쓰기 허용으로 설정해야 합니다.
